## 1. 무기 체계 대폭 개편
### AP / AD 개념 생성
AP 데미지는 마법류 데미지이다, 방어력을 관통하는 데미지류이고, 최대체력 증가로 저항한다.

AD 데미지는 일반적인 근접류 데미지. 방어력에 의해 감쇠되는 데미지류이고, 방어력 증가로 저항한다.

예를들어 최대체력만 높은 경우 높은 양의 근접 데미지는 견딜 수 없고,

방어력만 높은 경우, 방어력을 관통하는 마법류 데미지에 취약하다.

### 근접 무기군 
* 계수값은 검은 1이며, 각각의 의미는 검이 1일때, 검과 비례해서 어느정도의 데미지인지를 산출할 때 사용된다.

ex) 계수가 1.5인 무기가 있을때, 이와 같은 티어의 검 아이템의 데미지가 6일때, 이 아이템의 데미지는 6*1.5 = 9로 결정됨.

#### 검(Sword)
일반적인 무기. 기존 검계열 무기보다 공격속도가 버프되었다.

공격속도는 최소 1.6부터 2.0까지이다. 그러나 기존 검과는 다르게 광역 데미지 기능은 사라졌다. (실제 아이템은 괭이로 대체)

#### 대검류(GreatSword & Katana) 2.75
주 손에 들었을 시 속도감소 -20(+n)%

데미지가 가장 높고 공격속도는 0.6보다 높을 수 없다. 휘몰아치는 칼날 9레벨 인챈트가 자동적용(표기 데미지의 90%)된다.

*참고 : Attack_Damage × (Sweeping_Edge_Level / (Sweeping_Edge_Level + 1))*

#### 도끼 (Battleaxe) 1.4
기존 도끼계열 무기는 데미지는 준수하였으나 속도가 느렸지만, 이번에 속도가 일반적인 수준으로 향상되었다.

다만 데미지는 많이 너프 당하였으나 검보다는 높고, 속도도 검보다는 느린 정도. 공격 속도는 최소 1.0부터 1.2까지다.

특징으로는 '워 크라이'가 있다.  F를 눌러 마나를 소모하여 광폭화 모드가 되면, 10초 동안 저항1, 흡수1, 성급함2 버프를 얻게된다. 쿨타임은 고정 45초이다.

(구체적인 버프 강도,종류,지속시간은 무기에 따라 달라진다.)

#### 낫 (Scythe) 2.0
낫의 경우 근접 공격을 한 후 마법데미지 후속타가 들어온다. // 개발할때 무조건 AD딜링 데미지와 AP 딜링 데미지를 거의 절반 수준으로 만들어둘것.
속도는 해머보다 더 느린 0.6~0.8 사이이다.

#### 전쟁병기(전투해머/전투도끼) (WarWeapon \[Warhammer / WarAxe\]) 1.3
전쟁병기는 광역기 '지면강타'를 이용해 주변 광역 딜링에 특화된 무기이다.

F 를 눌러 지면 강타가 가능하고, 이때 주변에 존재하는 자신을 제외한 모든 엔티티는 스킬데미지를 입게된다.

광역 데미지는 4~10데미지. (커스텀 데미지시스템 이용)

기본 공격력은 소검과 비슷한 데미지이다. 속도는 0.8~1.0 사이이다.

스킬 사용시 : 0.3초간 이속3버프를 받게되고, 0.3초후에 주변으로 충격파를 날린다. 이후 1초간 이속-2에 걸린다. 

충격파데미지는 2\~12(엔드) 이다(클라우드울프의 커스텀 데미지 참고)

### 원거리 무기군
* 원거리 무기군은 기본적으로 주 데미지가 AP 데미지로 이루어져있다. 각 아이템별로 데미지가 별개로 설정되어 있다.

#### 활(Bow)
일반적인 활이다. 충전식으로 발사한다. 각 아이템마다 고유의 화살과 고유의 투사체(들)을 발사할 수 있다.

#### 석궁(Crossbow)
석궁으로 쏜 화살은 중력의 영향을 받지않고, 투사체 속도가 빠르다. (총과 거의 비슷하다고 보면 된다.)

장전속도는 느리고, 한번 장전해서 여러번 투사체를 발사할 수도 있다.

#### 투척형 단검(ThrowingKnife)
투척형 단검은 속도가 무한으로 빠르고 낮은 근접 데미지를 갖고 있는 투척형 원거리 무기이다.

#### 스태프(Staff)
스태프는 스태프만의 고유마법을 발사하는 발사 촉매이다.


## 2. 광물 시스템 대폭 개편
광물로 제작할 수 있던 아이템의 스텟이 대부분 개편되어 나온다.
너무 강력했던 방어구의 너프와 약했던 방어구의 버프가 이루어진다.
```
1티어 - 나무/돌
전체 웨폰셋이 존재하지않음

2티어 - 철
대검, (단검, 활) 만 추가적으로 존재

3티어 - 플래티넘, 다이아(3.2티어)
플래티넘은 플래티넘 광물을 채굴해서 얻는다. 다이아보다 얻기 쉽다.
다이아몬드 무기는 플래티넘 무기보다 약간 좋은 성능을 지녔다.

4티어 - 우는 흑요석
우는 흑요석을 연마하여 제작하는 무기.
흑요석 8개 + 가디언의 눈알 1개 = 우는 흑요석 8개
우는 흑요석 8개 + 엘더 가디언의 눈알 1개 = 우는 흑요석의 결정 8개
우는 흑요석의 결정으로 무기 제작.

5티어 - 크림슨 다이아몬드
피글린 브루트를 잡아 드랍되는 돼지의 피,
마그마 큐브를 잡아 얻는 마그마 크림으로
제조된 진홍빛 액체를 재료로 하여
네더 대장장이 작업대에서 다이아 무기를 '강화'하여 얻는 무기들이다.

참고) 크림슨 다이아몬드는 지옥에서 화폐로 사용됨.

6티어 - 네더라이트
지옥의 힘을 빌려 강화된 우민들 중, 파괴수와 그 외의 보스몬스터를 토벌하여 얻는 광물이다. 네더라이트로 네더 작업대에서 제작하여 얻을 수 있다.

7티어 - 이그니스 소울, 아스트라 소울
블레이즈 가루와 청금석을 촉매로 하여 소울과 함께 연성하여 네더라이트를 강화하여 얻는 광석들이다. 심연 작업대에서 조합하여 얻을 수 있다.

8티어 - 에이션트 어비셜
지옥에서 얻은 고대 잔해와 지옥에서 새로운 심연이라는 차원으로 이동해서 얻은 새로운 재료를 합성하여 얻은 광물. 전체적으로 모든 무기는 상대의 시야를 멀게함. 세트 갑옷 착용시 야간투시 효과, 를 받음

9티어 - 스타더스트 엔더마이트
어비셜 월드에서 입장 가능한 카오스 월드에서 카오스 위더를 잡아서 혼돈의 별을 보라보라 작업대에서 엔더진주와 주조하여 만든 광물이다. 이때부터 보스레이드 개념을 넣을 생각이다.
```

## 4. 경험치와 레벨개념 변화
청금석, 마법부여된 책은 구하기 어려워졌습니다. 그리고 무기 개명과 무기 수리를 위한 비용은 최소 레벨을 제외하고 모두 사라졌습니다.
청금석은 강화된 몬스터 사냥을 통해 획득할 수 있습니다.
마법이 부여된 책은 강화된 몬스터 사냥, 보스 몬스터 사냥을 통해 획득할 수 있습니다.
이때 보스 몹 사냥을 통해 획득하는 아이템은 기존 인챈트와는 다른 강력한 레벨의 인챈트가 부여된 책이 주어질 것입니다.

경험치는 이제 중요한 최대체력 수급요소로 변화되었습니다. 이 부분에 대해서는 난이도와 게임의 재미를 위해 자주 바뀌게 될 것 입니다.
```
10레벨 이전 : 하트 7개
60레벨 이전 : 하트 8개
100레벨 이전 : 하트 9개
140레벨 이전 : 하트 10개 (정상화)
180레벨 이전 : 하트 11개
180레벨 이후 : 하트 12개
```
그러나 경험치를 무작정 올리면 안됩니다.
100레벨 달성시 -> 월드레벨 1->2
200레벨 달성시 -> 월드레벨 2->3

완전히 위험하진 않지만, 더 강력한 적들도 나올 가능성이 농후해집니다.

### 4-1. 드랍 변화
몹 별로 경험치 드랍량이 변화됐습니다. -> 월드레벨 별로 몹 드랍 아이템에 경험치병, 청금석, 코인이 늘어납니다.

* 청금석, 경험치 병 등의 별도 드랍 아이템은 강화된 몹이 나올 수 있는 몬스터(적대적 몬스터)에 한해 드랍합니다.
* 몹타워 등의 자연사를 유도하여 몬스터를 죽일 시 드랍하지 않습니다. (killed_by_player가 인식 되어야 합니다.)

### 월드레벨1
코인 : 0 ~ 2개

경험치 병 : 2 ~ 3개 *(신중히 사용하십시오)

청금석 : 0 ~ 1개

### 월드레벨2
코인 : 1 ~ 3개

크림슨 다이아몬드 : 0 ~ 1개

경험치 병 : 3 ~ 6개

청금석 : 0.5 ~ 3개 (1/7 0개 2/7 1개 2/7 2개 2/7 3개)

### 월드레벨3
코인 : 4 ~ 8개

크림슨 다이아몬드 : 1 ~ 3개

어두운 엔더의 눈 : 0 ~ 1개

경험치 병 : 8 ~ 20개

인챈트북은 강화된 몹(또는 보스) 사냥 시 드랍합니다. 기존에 시행되던 낚시, 사서 노가다에서는 인챈트북 대신에 각각 청금석, 스펠북 레시피로 교체될 예정입니다.

낚시 > 인챈트북 -> 청금석 10개드랍 등
사서 > 인챈트북 -> 버프 주문서

사서를 달인 레벨 까지 올리면, 사서는 버프 주문서를 판매합니다.

버프 주문서는 1회용이므로, 신중하게 사용하도록 합니다.

### 월드레벨 1 사서(주민 사서)가 판매하는 버프 주문서
피의 맹세 (청금석10개 코인30개) : 40초간 공격력1 버프
굳건한 바람의 맹세 (청금석8개 코인15개) : 푸른 바람의 맹세. 30초간 공속1, 이속1 버프
### 월드레벨 2 사서(지옥 주민 사서)가 판매하는 버프 주문서
융해 : 40초간 이동속도 40% 저하, 방어력 40% 버프
맹독 : 공격하는 모든 공격에 독 속성 부여, 해당 독 속성은 4레벨 2초.
### 월드레벨 3 사서(심연 수호자 사서)가 판매하는 버프 주문서
어둠의 무기 : 공격하는 모든 공격에 시듦 속성 부여. 해당 속성은 2레벨 3초
심연의 가호 : 40초간 공격력2, 방어력1 버프.

## 전문기술 추가

자신이 할 수 있는 극한의 기술을 연마하여 전문기술 레벨을 올릴 수 있습니다.
전문기술을 통해 전용 작업대를 얻어 전용 무기를 얻어보십시오.
### 전문기술 목록
- 채광 -> 낫
- 벌목 -> 도끼
- 갑옷제작 -> 망치
- 무기제작 -> 대검, 검
- 인챈트 -> 스태프
- 스켈레톤 류 사냥 -> 활
- 우민 사냥 -> 석궁
- 에메랄드 수집(에메랄드를 바쳐서 업글) -> 단검

## 게임 시작 시 클래스 선택 가능
